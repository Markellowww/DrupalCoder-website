DrupalCoder is the final project of the "Website Development" course, a responsive website for a company specializing in Drupal development. The project demonstrates the application of fundamental front-end development knowledge.

### **Main Goal** 
To create a fully functional, responsive, and semantically correct website using standard web technologies.

### Technical Requirements

The following requirements were implemented within the project:
- Semantic layout using HTML5
- Responsive design (mobile-first approach)
- CSS Grid and Flexbox for layout
- Cross-browser compatibility
- Performance optimization
